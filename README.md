# Loan Default Prediction

A Flask-based Machine Learning web application that predicts whether a customer will default on a loan.

## Features
- User-friendly web interface
- Decision Tree model
- Real-time prediction

## Tech Stack
- Python
- Flask
- Pandas
- Scikit-learn
